# Jankenpon GRPC

Jankenpon GRPC Client-Server With Golang

How to use :

# For Server
1. ``` cd server ```
2. ``` go run server.go ```

# For Client
1. ``` cd client ```
2. ``` go run client.go ```

How to Play : 
1. Client always play first, input text ("Batu", or "Gunting, or "Kertas")
2. Server input and calculate the result.
